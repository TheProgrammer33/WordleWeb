Welcome to the wordle solver.

Word Length is number of letters that the word you're solving for has.
Green Letters and Yellow letters use the syntax t1 where t is the letter and 1 is the position in the word for that letter. So if the word starts with a t then you would put t1 in the green letters input. Or if t is not the first letter but it is in the word, then you would put t1 in the yellow input
Grey letters can be written like xzbr where all four of those letters are not in the word at all. Notes can be used however you want.
